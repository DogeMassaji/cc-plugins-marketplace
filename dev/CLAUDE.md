# dev

AI 编码 Agent 的工程工作流插件。覆盖软件开发生命周期：需求定义 → 任务规划 → 代码构建 → 审查修复。

## 工作方式

- **命令驱动**——slash command 触发各阶段工作流
- **Agent 专职化**——不同阶段由不同能力的 Agent 执行（策划、构建、审查）
- **严格串行**——各步骤顺序执行，失败即停

## 约定

- 所有面向人类的输出使用中文
- 代码标识符、技术术语、配置键保留原文
- 产物统一归档到 `.artifacts/<yyyymmdd>/<任务简述>/`

## 约束

- 不要添加模糊建议而非可执行流程的 Skill
- 不要在不同 Skill 间重复内容——引用其他 Skill

## 文档修改需要启用 caveman 进行压缩
1. 如下词不用过分压缩，如：每个 -> 每，完成 -> 完
